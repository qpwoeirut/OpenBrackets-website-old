Spring 2021: Intermediate Python Game Building


The Intermediate Python Game Building course is for students who have some experience with programming in Python. It is highly recommended that students take the Beginner Python Game Building class before this one. In order to be able to complete the course, students will need an internet connection, a device capable of running Zoom, and a computer that can access https://repl.it/. This course is taught online by the OpenBrackets teachers.


Lesson 1: Intro to Python Review Part 1

Week 1 Day 1

To refresh the skills learned in the Intro to Python class, students will review and practice each topic. The first day of the review will cover printing, libraries, variables, input, types, and conditionals.


Lesson 2: Intro to Python Review Part 2

Week 1 Day 2

This will be the second part of the Intro to Python review. It will cover the remaining topics, including loops, functions, lists, 2D lists, and dictionaries.


Lesson 3: Graphics

Week 2 Day 1

Students will be introduced to the Turtle graphics library in Python. We'll start by drawing basic shapes and designs.


Lesson 4: Turtle Race Part 1

Week 2 Day 2

In order to help students familiarize themselves with the Turtle Graphics library, we will build a game called Turtle Race. In Turtle Race, students will animate two turtles racing to a finish line, and then display the winner. This project will be finished in the next class.


Lesson 5: Turtle Race Part 2

Week 3 Day 1

Students will finish the Turtle Race game that they started in the previous lesson.


Lesson 6: Block Collector Part 1

Week 3 Day 2

Students will create a game called Block Collector. In Block Collector, a player moves a block around their screen and tries to collect other blocks to increase their score.


Lesson 7: Block Collector Part 2

Week 4 Day 1

Students will finish up their Block Collector game that was started in the lesson before.


Final Project

Week 4 Day 2 and Week 5

As a capstone project for the course, students will build the Snake game.
