Fall 2021: Beginner Python Game Building


This course covers the fundamentals of programming in Python. Students will learn about the applications of programming in Python and become familiar with the logic used in programming, including if/else statements, loops, and functions. Python containers like lists and dictionaries will also be covered. At the end of the course, students will use their new programming knowledge to create a game called Treasure Hunter.
No prior knowledge is required for this course, and everybody is welcome. In order to be able to complete the course, students will need an internet connection, a device capable of running Zoom, and a computer that can access https://repl.it/. This course is taught online by the OpenBrackets teachers.
