Fall 2021: Intermediate Python Game Building


This course builds on the Beginner Python course and covers Python graphics via a library called Turtle. Students will become familiar with how to use the Turtle library and build graphics-based games. As a part of this, students will learn about events and event handlers. At the end of the course, students will build their own versions of the classic arcade game Snake.
It is highly recommended that students take the Beginner Python Game Building or have prior experience with Python programming class before this one. In order to be able to complete the course, students will need an internet connection, a device capable of running Zoom, and a computer that can access https://repl.it/. This course is taught online by the OpenBrackets teachers.
