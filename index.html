Upcoming Offerings:


Python Game Design

This fall we will be offering a Game Design course that will allow students to learn the computer programming language of Python through simple games that they create after each lesson. During this 5 week course, students will learn how to tackle problems presented to them, the basics of Python, and will walk away with their own final game.

This course requires no previous Python or computer science knowledge. All you need is a computer and reliable internet connection. Check out some of what we will teach in this course under our Curriculum tab!

Game Design Juniors

This fall we will be offering a Game Design Juniors course that will allow students to learn the computer programming language of Scratch through simple games that they create after each lesson. During this 5 week course, students will learn how to approach a computer science problem and using blocks to code.
